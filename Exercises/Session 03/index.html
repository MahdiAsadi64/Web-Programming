جلسه سوم

جلسه سوم کلاس برنامه نویسی وب به تاریخ 28 دیماه 1402 برگزار گردید استاد پس از بررسی تمارین واگذاری شده ، درس را ادامه داده و موارد زیر در این جلسه تدریس فرمودند.

مفهوم attribute :کلمات ویژه ای هستند که مشخصات تگ های html را تعیین می کنند. به عنوان مثال :

  • align : تعیین میکند چیدمان آیتم در صفحه به شکل باشد ، راست چین ، وسط چین یا چپ چین.
  • src : در تگ img آدرس فایل تصویر را مشخص می کند
  • width : برای تعیین عرض یک آیتم استفاده می شود
  • height : برای تعیین ارتفاع آیتم استفاده می شود
  • color : برای تعیین رنگ آیتم استفاده می شود.
نکته 1 : عموما وظیفه چیدمان و ظاهر صفحه بر عهده css می باشد و کمتر از attribute استفاده می شود
نکته 2 : برخی attribute ها و همچنین برخی tagها در html منسوخ شده و به همین جهت رنگ آنها توسط vscode متفاوت از بقیه نمایش داده می شود.
نکته 3 : برای تعیین عرض یا ارتفاع می توان از واحد های px و یا % استفاده کرد. اولی بر حسب پیکسل و دومی بر حسب درصدی از اندازه والد می باشد.

تعیین ابعاد آیتم ها

در تصاویر زیر، عرض اولی برحسب پیکسل و معادل 150px و عرض دومی بر حسب درصد و معادل 50% تنظیم شده است و یه این ترتیب تصویر اولی همیشه و بدون توجه به ابعاد وادش دارای عرض 150 پیکسل می باشد اما دومی با تغییر اندازه والد دچار تغییر در ایعاد می شود
